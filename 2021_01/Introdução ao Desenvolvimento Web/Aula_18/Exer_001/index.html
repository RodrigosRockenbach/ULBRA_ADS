Calculadora Básica


Faça uma calculadora básica.

- dois inputs para capturar os números
- um select para capturar qual operação aritmética deverá efetuar
- um botão de calcular
- um label para o resultado.

Para pegar o conteúdo de um input utilize:
--> document.getElementById("idDoInput").value;

E para mostrar na tela (label) utilize
-->document.getElementById("idDoCampo").innerHTML = resultado...;

Utilize também o console.log( ) para acompanhar o que acontece no console.


Estória do usuário:

- o usuário digita o primeiro número
- o usuário escolhe qual operação (+, -, * ou /)
- o usuário digita o segundo número
- o usuário clica em calcular
- o sistema mostra o resultado na label de resultado
